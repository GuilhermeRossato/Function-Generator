Bezier Function Generator

The objective of this experiment is to generate multi purpose functions with interpolation. Using a modified Casteljau's Algorithm (inversed the equations), this application can generate custom functions for any purpose.

This experiment uses Javascript EMCA 5 and HTML5 Canvas element

Classes used:

  • Canvas Controller (interface/drawing control)
  • Circle (main objects)
  • GUI Box (handles alignment for buttons)
  • GUI Button (for interface)

Dependencies:

  • JavaScript Expression Evaluator by silentmatt
